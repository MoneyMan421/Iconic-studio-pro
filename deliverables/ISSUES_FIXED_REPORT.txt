======================================================
  ICONIC STUDIO PRO — ISSUES FIXED REPORT
======================================================

── Previously Fixed (pre-existing) ──────────────────

[FIX-1] test/widget_test.dart
  Tests now pump StudioPage directly, not AuthGate.
  Prevents async SharedPreferences load from blocking
  studio UI rendering in a single pumpWidget call.

[FIX-2] Stale shaders/lib/main.dart deleted
  File was causing ~30 flutter analyze errors.

── Fixed in this PR ──────────────────────────────────

[FIX-3] lib/main.dart → _buildStatsBar()  (CRITICAL)
  FPS was hard-coded to '120'. Replaced with a real
  Ticker-based frame counter (_fpsTicker) that measures
  actual rendered FPS every second and displays the
  live value in the stats bar.

[FIX-4] lib/main.dart → PaywallModal  (CRITICAL)
  "Upgrade Now" button was completely non-functional
  (no-op). Replaced with two distinct tier buttons
  (Pro Monthly $2.99/mo, Pro Lifetime $14.99).
  Each button closes the dialog and shows a SnackBar
  explaining that in-app purchase integration is
  upcoming (RevenueCat / Stripe hook clearly marked
  with TODO). No longer silently does nothing.

[FIX-5] lib/auth_screen.dart — AuthGate rebuild loop
  After a successful sign-up or login the gate state
  never refreshed, leaving the user stuck on the auth
  screen. Fixed via an onAuthenticated() callback
  passed from _AuthGateState through AuthScreen to
  _SignUpForm/_LoginForm so setState() fires on the
  gate itself.

[FIX-6] lib/auth_screen.dart — AppColors.error added
  Error text was using a hard-coded Color(0xFFCF6679).
  Added AppColors.error constant and updated all
  error-state Text styles to use it.

[FIX-7] deprecated withOpacity() removed throughout
  All colour opacity calls now use the correct
  withValues(alpha:) API to satisfy flutter analyze
  --fatal-infos.

[FIX-8] lib/editor_storage.dart — fully implemented
  Was a stub/skeleton. Now correctly persists and
  restores all EditorState fields (scale, rotation,
  brightness, contrast, saturation, blur,
  refractionIndex, sparkleIntensity, facetDepth) plus
  importsUsed via SharedPreferences.

[FIX-9] test/widget_test.dart — expanded coverage
  Added 8 meaningful widget/unit tests covering:
    • StudioPage renders without crash
    • AppBar action icons present
    • Upload zone shown when no image
    • Stats bar labels (FPS / IMPORTS / SCALE / ROTATE)
    • All control panel sections & slider labels
    • EditorState.copyWith field preservation
    • EditorState.copyWith clearImage flag
    • AppColors constants non-null
    • Reset button visibility
